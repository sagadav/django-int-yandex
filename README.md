### Запуск в Dev mode:

Создайте файл .env с переменнами окружения, пример в .env.example, далее:

```
pip install -r dev-requirements.txt
cd lyceumsite
py manage.py runserver
```

Переменные окружения:

```
SECRET_KEY - секретный ключ
```

Django v3.2
Python 3.11
