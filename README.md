### Запуск в Dev mode:

Создайте файл .env с переменными окружения (пример: .env.example), далее:

```
pip install -r dev-requirements.txt
cd lyceumsite
py manage.py runserver
```

Переменные окружения:

```
SECRET_KEY - секретный ключ
```

Django v3.2
Python v3.11
